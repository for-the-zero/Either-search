# Either-search
###### only Chinese Ver.

---

爬虫Google、bing、qwant搜索结果
通过算法整合到一起

只有简单的链接，顶多是镜像提示和哔哩哔哩视频浏览

---

## 使用方法
1. 安装pypi：lxml、requests、pywebio（好像就这些了）
2. 运行run.py
3. 自动打开浏览器，输入关键词搜索
4. 等待
5. 出来结果，只有链接
